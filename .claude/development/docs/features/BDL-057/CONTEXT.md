## Key decisions (locked in PRD/RFC; owner-approved)

- **Layer 1 = blocking** in `beadloom ci` on `stale>0`. Layer 2 = **warn only**. (owner-approved)
- **Layer 2 declaration** = in-doc annotation `<!-- beadloom:watches=cli,graph,flow.yml -->`.
- **Watched surfaces** are coarse identity signatures: `cli` (Click command+flag tree), `graph` (node+edge identity set), `flow.yml` (normalized config). Aggregate hash = hash of concatenated signatures in declared order.
- **Baseline** in a NEW `reference_state` table parallel to `sync_state` (additive; symbol-pair logic + reason-masking untouched).
- Reference drift surfaces in `sync-check` as `reason = surface_drift`, severity **warning**; cleared by `sync-update <doc>`.
- **G4 in-scope** (owner-confirmed): fill the 11 skeleton SPECs + declare `watches` on README(en/ru)/getting-started/architecture, in this feature.
- No public «экосистема»/"ecosystem". Documentation follows the BDL-056 tech-writer writing-quality standard.
