## Decisions (from PRD/RFC)

- **Preserve the flow 1:1.** Scaffold templates = byte-identical vendored copies of the live `.claude/agents/*` + `commands/*`; a drift-guard test enforces equality. No rewrite/condense (owner's hard requirement).
- **`setup-agentic-flow` command** (setup-* family), not an `init` flag (optional alias may delegate).
- Per-project facts (version/stack/packages/CLI) stay in the **CLAUDE.md auto-regions**, generated by `config_sync` — never hardcoded into the vendored flow files.
- **MCP = deterministic process-tools, not orchestration.** Coordinator + Agent-spawn stay harness/Claude-Code-native (G4). `complete_bead` is advisory-strong; true enforcement remains `beadloom ci` in CI (G5).
- **Tools-only v1** (MCP prompts deferred).
- `bd` invoked via a thin **mockable seam** (testable without `bd`; clear error if absent).

## Code standards (from CLAUDE.md §0.1)

- **Stack:** Python 3.10+, SQLite, Click, Rich, tree-sitter.
- **Tests:** pytest + pytest-cov (coverage ≥ 80% on changed code).
- **Lint/format:** ruff. **Types:** mypy --strict (no `Any` / `# type: ignore` without reason).
- **Architecture:** DDD packages — `application/`, `context_oracle/`, `doc_sync/`, `graph/`, `infrastructure/`, `onboarding/`, `services/`, `tui/`. Respect boundaries (`beadloom lint --strict`).
- No bare `except`, no `import *`, no mutable default args, no `print()`/`breakpoint()`.
- New code: `setup-agentic-flow` → `onboarding/`; MCP tools → `services/mcp_server.py`; templates → `onboarding/templates/agentic_flow/`.

## Constraints / invariants

- Beadloom stays green on its own `beadloom ci` (lint/sync-check/config-check/doctor) + full pytest after every wave.
- The scaffold must be **idempotent** + never touch user prose outside auto-regions.
- The drift-guard ties the vendored templates to the live `.claude/` — re-vendor when the live flow changes.
- MCP tools deterministic + unit-tested with `bd` + gate **mocked** (no network/external `bd`).
- Anonymize any third-party project names in committed artifacts.
